Bookshelf is a web application built using Python, Flask, and Bootstrap that allows users to search, explore, and review books. The platform provides an intuitive interface to browse a collection of books sourced from a Kaggle dataset, with detailed information and interactive features for users to engage with.


Key Features

•	Book Search Functionality:
Users can search for books based on various parameters such as title, author, or year of publication.

Example: Searching by year “2000” displays all books published in that year in a clean, responsive tabular layout.

•	Dynamic Results Display:
The search results are presented in a Bootstrap-styled table, making it easy to browse and compare books.

•	Individual Book Page:
Clicking on a specific book in the results table navigates to a dedicated book detail page, which displays:

o	Book cover image
o	ISBN
o	Title and author
o	Year of publication
o	Description and other metadata

•	User Authentication:
The application includes a login and registration system, enabling users to securely log in to their accounts.

Technologies Used
•	Backend: Python, Flask
•	Frontend: HTML, CSS, Bootstrap
•	Database: SQLite
•	Data Source: Book dataset from Kaggle

•	Book Reviews:
Logged-in users can write and submit reviews for individual books, allowing for an interactive and community-driven experience.
